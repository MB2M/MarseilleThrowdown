AMRAP - 9mins

Individual scorecard
There is a scaled version of this workout:
  • 8 Wall Walk = 8m Handstand Walk
  • Complex = 3 PU + 4 T2B
Scaled verison of the workout will be ranked behind RX version except if RX doesn't end the first round.
